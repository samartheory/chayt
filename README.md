
# Chayt

A room based chatting web app. Built with ReactJS, NodeJS, Express and Socket.io




## Deployment

To run this project follow these steps

1. Run the server in the server folder 
```bash
  nodemon index.js
```
 
2. Start the React App in the Client folder
```bash
  npm start
```
